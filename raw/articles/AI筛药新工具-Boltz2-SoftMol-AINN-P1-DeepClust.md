# AI筛药新工具：DIAMOND DeepClust / Boltz-2 / SoftMol / AINN-P1

**来源：** 榴莲忘返 AIDD（微信公众号）

---

## 1. DIAMOND DeepClust：36倍加速蛋白质聚类

- **级联聚类**（先粗后精）：线性算法快速聚类相似序列 → 逐步提高比对灵敏度
- NCBI NR（5.46亿序列）：DeepClust **19小时** vs MMseqs2 **29天**（快36倍）
- 灵敏度68.6% vs 62.3%，精度95.5%
- **190亿序列**聚类为17亿簇，5.44亿非单例簇覆盖94%序列
- 集成AlphaFold2：低覆盖度蛋白pLDDT从52.9提升至62.6

## 2. Boltz-2：共折叠模型虚拟筛选重打分

- 蛋白质-配体 **共折叠** 策略，两者相遇后动态调整
- ULVSH基准（10个真实对接项目）：**ROC-AUC 0.70**，10个靶点中成功7个
- 速度：单GPU约100秒/配体，千级分子一天完成
- **反直觉发现**：打分准确性与结合模式准确性无直接关联——模式错了（RMSD>2Å）但仍能正确区分活性/非活性
- 实际应用：超大规模对接后，**富集率提升4-5倍**
- 局限：2个靶点（CNR1/MTR1A）表现比瞎猜还差（ROC-AUC 0.40-0.45）

## 3. SoftMol：块扩散分子生成新范式

- 从"逐token拼写" → **"软片段搭积木"**
- SoftBD架构：块内双向纠错 + 块间顺序搭建
- **100%有效性**，采样速度比扩散模型快**6.6倍**
- MCTS目标导向生成 + **可行性门**预筛选（QED/SA打分）
- 精选ZINC-Curated数据集（4.27亿）：类药性94%
- 5靶点刷新纪录，top 5%对接分数提升**9.7%**，新化学骨架多2-3倍

## 4. AINN-P1：小模型高效预测蛋白稳定性

- **mLSTM代替Transformer**：计算复杂度从平方级降到线性
- 仅1.67亿参数，仅需序列（无需MSA/3D结构）
- ProteinGym稳定性预测Spearman ρ最高 **0.637**
- 定位：**前端高速过滤器**，几十万序列初筛→缩小到几百→再交给精细模型
- 最佳实践：全局性质用mean-pooling + 少量数据用岭回归
