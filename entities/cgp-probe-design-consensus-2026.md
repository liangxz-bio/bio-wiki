# 基于NGS的肿瘤全景变异检测探针设计专家共识（2026版）

> 发布：中国抗癌协会肿瘤病理专业委员会 / 中华肿瘤杂志 2026, 48(4): 491-501
> DOI: 10.3760/cma.j.cn112152-20250722-00356
> 技术参与方：吉因加 (Geneplus)
> 牵头单位：国家癌症中心 / 中国医学科学院肿瘤医院 / 中检院 / 复旦大学附属肿瘤医院

中国首个肿瘤全景变异检测（CGP）探针设计专家共识，填补了探针设计领域行业标准空白。与此前收录的 [[gbt46943-2025-mngs-standard]]（mNGS 性能确认）互补，两者分别规范 CGP 探针设计和 mNGS 性能确认，构成 NGS 检测质量标准的两个核心维度。

## 六大共识要点

### 共识一｜基因变异分级标准及基因证据等级分类体系（2A类）

| 类别 | 定义 | 纳入建议 |
|------|------|----------|
| I 类基因 | 具有明确临床意义 | 必须纳入 |
| II 类基因 | 具有潜在临床意义 | 推荐纳入 |
| III 类基因 | 临床意义未明 | 战略性纳入，维持产品全生命周期竞争力 |

### 共识二｜探针池目标区域设计（2A类）

探针池由两大模块组成：

```
探针池目标区域
├── 检测功能模块
│   ├── 基因变异检测靶区：SNV / InDel / CNV / SV
│   └── 基因组指标靶区：TMB / MSI / HRD
└── 质量控制模块
    ├── 肿瘤纯度推算
    ├── 样本配对分析
    ├── 污染率评估
    └── 性别判断
```

每个模块需按产品预期用途个性化设计。

### 共识三｜常见实体瘤基因检测推荐列表（1类）

综合考量三个维度：
- **临床预期用途**：辅助诊断 / 疗效预测 / 预后评估
- **肿瘤来源系统**：消化 / 妇科 / 中枢神经等
- **特定治疗方式**：免疫治疗（TMB/MSI）、PARP 抑制剂（HRD/BRCA）

支持多癌种或泛癌种基因检测产品设计。

### 共识四｜探针设计与筛选规范（2A、2B类）

| 参数 | 推荐范围 | 推荐级别 |
|------|----------|----------|
| GC 含量 | 50%–60% | 2A 类 |
| 探针长度 | 60–120 nt | 2A 类 |
| 交叉杂交阈值（50mer） | 相似性 > 75%–80% | 2A 类 |
| 人群多样性考量 | 参考基因组 + 人群遗传多样性 + 特定突变区域 | 2B 类 |

### 共识五｜探针池性能评估体系（1类）

三层评估框架：

```
第一层：质控评估
├── 捕获效率
├── 均一性（Fold80）
└── GC偏好

第二层：单条探针/单个位点评估
└── 深度系数 / 覆盖度达标

第三层：产品维度检测能力评估
├── LoD（检出限）
└── 准确度（PPA / NPA / PPV ≥ 95%）
```

### 共识六｜探针变更管理规范（2B类）

| 变更情形 | 管理要求 |
|----------|----------|
| 仅探针池变更 | 质控表现 + 深度系数达标，更新生信流程 |
| 供应商变更 | 与变更前探针进行全变异类型准确度对比 |
| 目标区域新增/删减 | 重新确认数据量 / LoD / 准确度 |
| 部分探针浓度调整 | 重新确认数据量 / LoD / 准确度 |

## 关键参数速查

- **Fold80** 均一性阈值：未给出具体数值，需结合产品预期用途设定
- **PPA/NPA/PPV ≥ 95%**：与 [[gbt46943-2025-mngs-standard]] 的 ≥95% 符合率一致
- **肿瘤纯度推算**：探针池需包含用于纯度估算的 SNP 位点
- **生信流程同步更新**：探针变更后必须更新生信流程
