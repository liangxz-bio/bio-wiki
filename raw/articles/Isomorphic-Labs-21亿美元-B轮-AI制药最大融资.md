# 143亿元！AI制药最大单笔融资！Isomorphic Labs B轮21亿美元

**来源：** 写意宣发（微信公众号，作者 写意宣发）

---

2026年5月12日，从谷歌DeepMind分拆而出的AI药物研发公司 **Isomorphic Labs** 宣布完成 **21亿美元（约合人民币143亿元）B轮融资**，创下AI制药领域单轮融资的最高纪录。

## 融资细节

| 维度 | 内容 |
|:---|:---|
| **公司** | Isomorphic Labs（Google DeepMind孵化） |
| **融资金额** | **21亿美元**（约143亿元人民币） |
| **轮次** | B轮 |
| **领投方** | Thrive Capital（连续第二年领投） |
| **跟投方** | Alphabet、GV（老股东） |
| **新进者** | MGX（阿布扎比）、淡马锡（新加坡）、英国主权AI基金 |
| **A轮对比** | 2023年A轮仅6亿美元，B轮翻了**三倍多** |
| **估值** | 未披露 |

## 背景

Isomorphic Labs 由谷歌DeepMind创始人、诺奖得主 **Demis Hassabis** 于2021年创立，专注于将AlphaFold等AI技术应用于药物发现。本轮融资的规模刷新了AI制药领域的单笔融资纪录，反映资本市场对AI驱动药物研发的高度信心。
