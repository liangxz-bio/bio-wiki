## 1. 初识DeepVariant

DeepVariant是一个基于深度学习的**variant calling**软件，DeepVariant支持在二倍体生物中进行胚系变异调用，它的工作流程有三步：

> 注：此图片来源于 https://github.com/google/deepvariant

**第一步：** BAM/CRAM文件作为输入，将比对结果生成图片

**第二步：** 卷积神经网络识别图片，辨别变异（我的理解是：有一个被训练好了的机器人，能够识别出图片中的不同之处，并将这些不同进行分类）

**第三步：** 输出 VCF 或 gVCF 文件和 HTML 报告文件

## 2. DeepVariant安装——docker

```bash
BIN_VERSION="1.6.1"

sudo apt -y update
sudo apt-get -y install docker.io
sudo docker pull google/deepvariant:"${BIN_VERSION}"
```

## 3. DeepVariant 运行

```bash
docker run \
  -v ~/t60/pop_analysis:/work google/deepvariant:1.6.1 \
  /opt/deepvariant/bin/run_deepvariant \
  --model_type=WGS \
  --ref=/work/data/hap.fa \
  --reads=/work/map/A.dedup.sort.bam \
  --output_gvcf=/work/deepvariant/A.g.vcf.gz \
  --intermediate_results_dir \
  /work/deepvariant/intermediate_results_dir \
  --num_shards=1
```

> `-v` 参数设置的是工作目录，即进行群体遗传学分析的文件夹目录（`~/t60/pop_analysis`）。`~/t60/pop_analysis:/work` 的意思是指在docker运行中work目录等于 `~/t60/pop_analysis`，所以在 `--reads`、`--output` 等文件夹所在目录都以 `/work` 开头，即 `/work`（`~/t60/pop_analysis`）下面的目录。

生成结果文件 `A.g.vcf.gz` 之后进行文件转化，挑选SNP变异以及过滤：

```bash
gatk GenotypeGVCFs -R Hap.fa \
  -V A.g.vcf.gz -O A.vcf.gz -new-qual True
# .g.vcf.gz 转化为 .vcf.gz 文件

gatk SelectVariants -select-type SNP \
  -V A.vcf.gz -O A.snp.vcf.gz
# 将SNP变异挑选出来

gatk VariantFiltration -V A.snp.vcf.gz \
  -O A.snp.filter.vcf.gz \
  --filter-expression "DP < 8.0 || DP > 200.0 || QD < 2.0 \
  || MQRankSum < -12.5 || ReadPosRankSum < -8.0 \
  || MQ < 40.0 || FS > 60.0 || SOR > 3.0" \
  --filter-name "Filter"
# 过滤变异
```

> 具体参数设置看 **群体遗传学分析 | 4.1 GATK—SNP calling**

## 参数说明

DeepVariant `run_deepvariant` 参数：
| 参数 | 说明 |
|------|------|
| `--model_type=WGS` | 全基因组测序模式 |
| `--ref` | 参考基因组文件路径 |
| `--reads` | 比对后的BAM文件 |
| `--output_gvcf` | 输出gVCF文件 |
| `--intermediate_results_dir` | 中间结果目录 |
| `--num_shards` | 并行分片数 |

GATK后处理参数：
| 工具 | 用途 |
|------|------|
| `GenotypeGVCFs` | gVCF → VCF 基因分型 |
| `SelectVariants -select-type SNP` | 筛选SNP |
| `VariantFiltration` | 变异质量过滤 |

VQSR过滤条件：
| 条件 | 阈值 | 说明 |
|------|------|------|
| DP | 8.0–200.0 | 覆盖深度 |
| QD | ≥ 2.0 | 变异质量/深度比值 |
| MQRankSum | ≥ -12.5 | 比对质量的秩和检验 |
| ReadPosRankSum | ≥ -8.0 | 读取位置秩和检验 |
| MQ | ≥ 40.0 | 比对质量 |
| FS | ≤ 60.0 | Fisher检验链特异性 |
| SOR | ≤ 3.0 | 链偏向性比值 |
