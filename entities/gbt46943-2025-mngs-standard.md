# GB/T 46943-2025 — 临床实验室检测和体外诊断系统 病原宏基因组高通量测序性能确认通用要求

> 发布：国家市场监督管理总局 / 国家标准化管理委员会，2025
> 提出：国家药品监督管理局
> 归口：全国医用临床检验实验室和体外诊断系统标准化技术委员会 (SAC/TC 136)
> 起草：北京协和医院、中检院、华西医院、中山一院等 20+ 家单位

中国首个 mNGS 临床检测性能确认的国家标准。适用于临床检验实验室开展 mNGS 前的性能确认，**不适用于 tNGS（靶向探针杂交捕获/多重引物扩增）和纳米孔单分子测序**。

## 适用范围

| 包含 | 不包含 |
|------|--------|
| mNGS 乌枪法宏基因组测序 | tNGS (靶向探针杂交捕获) |
| Illumina 等短读长 NGS 平台 | tNGS (多重引物扩增) |
| 病原微生物检测 | 纳米孔/固态纳米孔单分子测序 |

## 性能确认架构

### 4.1 性能确认标本的制备

三种标本类型及其用途：

| 标本类型 | 定义 | 用途限制 |
|----------|------|----------|
| 临床标本 | 真实临床剩余标本 | 金标准，推荐首选 |
| 模拟阳性标本 | 阴性临床标本掺入定量病原体 | 不可用于建立临界值、准确度性能确认 |
| 计算机模拟测序数据 | 模拟 reads（打断→模拟测序→掺入真实阴性数据） | 仅用于生信流程性能确认 |

### 4.2 关键试验环节的性能确认

| 环节 | 关键确认内容 |
|------|-------------|
| 标本采集运输及保存 | 温度/时长/冻融次数稳定性，≥95%符合率 |
| 核酸提取试剂 | 产量/纯度/完整度，重复性+批间差异 |
| 文库制备试剂 | 片段长度分布/纯度/浓度/总量，重复性+批间差异 |
| 测序仪器及试剂 | 测序覆盖率/平均深度/准确率/重复性 |
| 生物信息学分析流程 | **临界点建立**（ROC曲线+约登指数）、物种鉴定能力（准确率/召回率/精确率/F1）、近缘物种交叉率 |

### 4.3 完整流程的性能确认

| 性能指标 | 评估方法 | 判定标准 |
|----------|----------|----------|
| 精密度（重复性+中间精密度） | 阴阳性标本 ≥2 批次 × ≥2 重复/批次 | 阴阳性符合率 ≥95% |
| LoD（检出限） | 梯度稀释 + 重复检测 → probit 分析 → 95% 阳性检出率浓度 | LoD95% |
| 稳定性 | 试剂冻融/保存条件测试 | 符合率 ≥95% |
| 分析特异性：干扰试验 | 加干扰物 vs 不加的阳性/弱阳性标本 | 干扰程度定量 |
| 分析特异性：交叉反应 | 近缘物种不同比例混合 | 交叉率定量 |
| **准确度** | 参照方法（培养/一代测序/临床诊断）并行检测 ≥**30** 阳性 + ≥**50** 阴性标本 | 敏感性 + 特异性 |

### 4.4 性能确认报告

必须包含：
- 预期用途
- 检测系统（测序仪/试剂/生信软件版本/数据库版本/标本类型/病原谱/SOP）
- 原始数据（只读格式保存，注明位置）
- 局限性（病原体层面 + 标本类型层面）

## 对现有 mNGS/tNGS 项目的影响

1. **KCQ 定量系统**（[[kingcreate-kcq]]）：标准要求 LoD95% 验证，KCQ 的绝对定量可作为参考方法
2. **RP100 tNGS panel**（[[kingcreate-tngs-panel]]）：标准明确排除了 tNGS，但准确度部分的方案（≥30阳性+≥50阴性）可参考
3. **临床评价**：标准的准确度要求与王辉团队 EBioMedicine 2024 研究的 251 例验证设计一致
4. **生信流程**：标准明确要求临界点通过 ROC 曲线建立、F1 度量评估物种鉴定能力，与 nf-core 流程的评估指标对接
